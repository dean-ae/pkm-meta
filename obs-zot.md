Integrate Obsidian & Zotero for academic note-taking and writing.

## Steps

1. Download [Obsidian](https://obsidian.md/) and [Zotero](https://www.zotero.org/).
2. Install Obsidian community plugins:
	- Pandoc Reference List
	- Zotero Integration
3. Install Zotero extensions:
	- Better BibTex
	- (optional) ZotMoov
